# Project Summary
The project is a web application developed for Zuru Web Solutions, designed to enhance user interaction and showcase services through a marketing platform. It integrates both frontend and backend functionalities, featuring user authentication, data storage, and a flexible API for custom backend capabilities.

# Project Module Description
The project consists of two main modules:
- **Frontend**: Built with modern web technologies, it delivers an interactive user interface and reusable components.
- **Backend**: Provides database functionalities, user authentication, and API endpoints, supporting various features and future enhancements.

# File Description Inventory
- **app/backend/README.md**: Overview and setup instructions for the backend.
- **app/frontend/README.md**: Overview and setup instructions for the frontend.
- **app/backend/main.py**: Main application entry point for backend services.
- **app/frontend/src/components/**: Contains reusable UI components.
- **app/backend/routers/**: Defines API endpoints for the application.
- **app/frontend/package.json**: Lists frontend dependencies and scripts.
- **app/frontend/pnpm-lock.yaml**: Lock file for managing frontend dependencies.
- **app/frontend/src/pages/AuthCallback.tsx**: Handles authentication callback and error management.
- **app/frontend/src/components/Hero.tsx**: Displays the hero section with improved layout and visibility.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic for database migrations

# Usage
To set up the project, follow these steps:
1. Install frontend dependencies:
   ```bash
   cd /workspace/app/frontend && pnpm install
   ```
2. Install backend dependencies:
   ```bash
   cd /workspace/app/backend && pip install -r requirements.txt
   ```
3. Run the backend application:
   ```bash
   python main.py
   ```
4. Run the frontend application:
   ```bash
   npm run start
   ```
